# Usage of simple-ssh:
#### example of usage ./simple-ssh -h 192.168.1.10 -u user    	
    	
##### ./simple-ssh --help
  	
  | Argument       | Type | Assignment of the argument | Default |
  | ------------- | ------------- | ------------- | -------------|
  | -h  | String                  | Host          |              |
  | -p  | Int                     | Port          | 22           |
  | -pk | String                  | Private key   | $HOME/.ssh/id_rsa |
  | -u | String                   | User name     |             |
